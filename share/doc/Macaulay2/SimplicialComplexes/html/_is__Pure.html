SimplicialComplexes :: isPure

isPure -- whether the facets are equidimensional

Synopsis

  • Usage:
    isPure D
  • Inputs:
    • D, a simplicial complex
  • Outputs:
    • a Boolean value, true if the facets of D all have the same dimension, and false otherwise

Description

i1 : R = ZZ[a..f];
 
i2 : D = simplicialComplex {a*b*c, a*b*d, d*e*f} 

o2 = | def abd abc |

o2 : SimplicialComplex
 
i3 : isPure D

o3 = true
 
i4 : E = simplicialComplex {a*b*c, b*d, d*e*f} 

o4 = | def bd abc |

o4 : SimplicialComplex
 
i5 : isPure E

o5 = false
