SimplicialComplexes :: substitute(SimplicialComplex,PolynomialRing)

substitute(SimplicialComplex,PolynomialRing) -- Substitute a simplicial complex to a different ring.

Synopsis

  • Usage:
    substitute(C,R)
  • Function: substitute
  • Inputs:
    • C, a simplicial complex
    • R, a polynomial ring
  • Outputs:
    • a simplicial complex

Description

Substitute a simplicial complex to a different ring. R should contain the variables of the ring of C.

i1 : K=QQ;
 
i2 : R=K[x_0..x_4];
 
i3 : I=monomialIdeal(x_0*x_1,x_1*x_2,x_2*x_3,x_3*x_4,x_4*x_0);

o3 : MonomialIdeal of R
 
i4 : C=simplicialComplex I

o4 = | x_2x_4 x_1x_4 x_1x_3 x_0x_3 x_0x_2 |

o4 : SimplicialComplex
 
i5 : S=R**K[y]

o5 = S

o5 : PolynomialRing
 
i6 : C1=substitute(C,S)

o6 = | x_2x_4 x_1x_4 x_1x_3 x_0x_3 x_0x_2 |

o6 : SimplicialComplex
 
i7 : ring C1

o7 = S

o7 : PolynomialRing
