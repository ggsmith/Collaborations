SimplicialComplexes :: useFaceClass

useFaceClass -- Option to return faces in the class Face

Description

Boolean Option to return in the methods faces and facets a List of Faces instead of a Matrix.

Functions with optional argument named useFaceClass :

  • faces(..., useFaceClass => ...)
  • facets(..., useFaceClass => ...)

For the programmer

The object useFaceClass is a symbol.
